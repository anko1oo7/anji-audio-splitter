# 🚀 Anji Universal Audio Splitter

ブラウザ上で簡単・高速・精密に音源を切り分けられるWebツールです。
15秒のショート動画素材から、自由な長さのサンプリングまで、マウス操作だけで完璧にコントロールできます。

## ✨ 特徴
- **波形ビジュアライザー**: 音の形を見ながら、ミリ秒単位で「ここだ！」というポイントを特定できます。
- **自由な切り出し設定**: デフォルト秒数を自由に変更でき、さらにマウスドラッグで範囲を微調整可能です。
- **プライバシー保護**: すべての処理はあなたのブラウザ内（Web Audio API）で行われます。音源データがサーバーに送信されることはありません。
- **一括ダウンロード**: 複数の切り出し範囲を一度に作成し、連番ファイルとしてまとめて保存できます。
- **プレミアムデザイン**: ダークモードとガラスモーフィズムを採用した、美しく直感的なインターフェース。

## 🚀 使い方
1. [index.html](https://anko1oo7.github.io/anji-audio-splitter/) を開きます。
2. 音源ファイル（MP3, WAV, M4A等）をドラッグ＆ドロップします。
3. 波形を見ながら、切り出し範囲（ピンクの領域）を調整したり、追加したりします。
4. 「切り出し実行」をクリックして、生成されたセグメントを確認・保存します。
